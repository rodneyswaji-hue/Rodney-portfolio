# Rondey Swaji — Creative Web Developer

This is a modern, minimal portfolio built with React + Vite + Tailwind CSS and Framer Motion. It’s designed to be lightweight, easy to extend and deploy to Vercel.

## Quick setup
1. `npm install`
2. `npm run dev`
3. Visit `http://localhost:5173`

## Deploying
- Connect this repository to Vercel and set the build command to `npm run build` and the output directory to `dist`.

## Notes
- Replace the placeholder email in `Contact.jsx` with your real email or contact flow.
- Update project links to your real repos and live demos.
- Add images in `public/` and reference them in the project cards for visual polish.
